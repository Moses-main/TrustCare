# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of TrustCare
- Patient dashboard with health metrics
- Provider dashboard for patient management
- Medical records management
- Appointment scheduling system
- Blockchain-based access control
- IPFS file storage integration
- Email authentication via Privy
- Wallet authentication via Privy
- Role-based access control (Patient, Provider, Admin)
- JWT-based session management

### Frontend Features
- React 18 with Vite
- Tailwind CSS styling
- Responsive design
- Patient portal
- Healthcare provider portal
- Landing page
- Profile settings

### Backend Features
- Express.js REST API
- MongoDB database
- Authentication system
- Medical records API
- Appointments API
- Patient management API
- Provider management API

### Blockchain Features
- HealthcareRecords main contract
- AccessControl contract
- PatientRecords contract
- ProviderManagement contract

### Removed
- Google authentication (email and wallet only)
- SendGrid email verification

## [0.0.1] - 2024-01-01

### Added
- Project initialization
- Basic project structure
